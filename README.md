## ✨ 功能特性

| 功能 | 描述 |
|------|------|
| 🎯 **智能订单筛选** | 根据物品易碎属性与目标接收点进行交叉筛选 |
| 🖱️ **自动化操作** | 全自动订单接取、翻页、刷新，解放双手 |
| 🔍 **视觉识别** | 基于 PaddleOCR 的屏幕图像识别，不修改游戏内存 |
| 🛡️ **防呆机制** | 自动处理委托失败弹窗，平滑过渡到下一订单 |
| ⚡ **智能滑动** | 支持鼠标拖拽/滚轮滚动，自动反转方向最大化效率 |
| 🖼️ **悬浮窗界面** | 实时状态显示，可随时暂停/恢复任务 |

## 🔧 核心机制

```
┌─────────────────────────────────────────────────────────────┐
│                      屏幕捕获层                              │
│                    (实时游戏画面)                            │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    PaddleOCR 识别引擎                        │
│              (文字提取 → 订单信息解析)                        │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    条件匹配过滤器                            │
│         (易碎属性 ✓ + 目的地 ✓ = 目标订单)                    │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                   PyAutoGUI 执行层                           │
│            (坐标定位 → 鼠标点击/拖拽 → 任务接取)              │
└─────────────────────────────────────────────────────────────┘
```

> **安全声明**：本工具完全基于屏幕视觉识别与物理级模拟键鼠操作，**绝不修改游戏内存或注入代码**。

## ⚙️ 环境配置

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| Python 版本 | 3.8 ~ 3.11 |
| 屏幕分辨率 | 1920×1080 或更高（推荐） |
| 权限要求 | 管理员权限（自动申请） |

### 依赖安装

```bash
# 1. 基础依赖
pip install pyautogui opencv-python tkinter

# 2. OCR 引擎（CPU 版本）
pip install paddlepaddle -i https://mirror.baidu.com/pypi/simple
pip install paddleocr

# 3. 打包工具（可选）
pip install pyinstaller
```

### DPI 缩放兼容

本工具已内置 **DPI 缩放感知屏蔽**，支持 100% ~ 200% 系统缩放比例，无需手动调整。

## 🚀 使用指南

### 启动流程

```bash
python main.py
```

1. **权限确认** → 程序自动请求管理员权限（UAC 弹窗）
2. **界面加载** → 悬浮窗显示于屏幕最上层
3. **参数配置** → 设置易碎信息、目的地、识别间隔等
4. **开始运行** → 界面最小化，后台监控启动

### 界面参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 易碎属性 | 筛选物品的易碎等级 | 根据需求设定 |
| 目的地 | 目标接收点名称 | 当前需要运送的据点 |
| 识别间隔 | 两次屏幕扫描的间隔时间 | 0.5 ~ 1.0 秒 |
| 刷新等待 | 列表刷新后的等待时间 | 1.0 ~ 2.0 秒 |
| 滑动模式 | 鼠标拖拽 / 滚轮滚动 | 根据模拟器选择 |

### 快捷操作

| 操作 | 说明 |
|------|------|
| `点击开始` | 启动自动化循环 |
| `点击停止` | 安全终止当前任务 |
| `任务栏呼出` | 随时查看运行状态 |
| `最小化界面` | 避免遮挡游戏画面 |

> ⚠️ **重要提示**：程序接管鼠标期间，请勿频繁手动操作，以免干扰自动化流程。

## 📦 打包部署

### 生成独立可执行文件

```bash
# 安装打包工具
pip install pyinstaller

# 一键打包（无控制台窗口）
pyinstaller -F -w -i icon.ico main.py
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `-F` | 打包为单个 .exe 文件 |
| `-w` | 无控制台窗口模式 |
| `-i icon.ico` | 自定义程序图标（可选） |

### 打包优化

- ✅ 标准输出/错误流黑洞重定向
- ✅ 第三方进度条库无终端兼容
- ✅ 自动提权检测内置

## ⚠️ 注意事项

| 类型 | 说明 |
|------|------|
| 🔐 **权限** | 必须以管理员身份运行，否则无法控制游戏窗口 |

## ❓ 常见问题

<details>
<summary><b>Q: 程序启动后闪退/无响应？</b></summary>

**A:** 请检查：
1. Python 版本是否在 3.8~3.11 范围内
2. 是否以管理员身份运行
3. 依赖库是否完整安装（`pip list` 核对）
</details>

<details>
<summary><b>Q: 鼠标点击位置偏移严重？</b></summary>

**A:** 可能是 DPI 缩放问题：
1. 检查系统缩放比例（设置 → 显示 → 缩放）
2. 右键程序 → 属性 → 兼容性 → 更改高 DPI 设置 → 勾选"替代高 DPI 缩放行为"
</details>

<details>
<summary><b>Q: OCR 识别不准确/漏识别？</b></summary>

**A:** 尝试以下优化：
1. 增加识别间隔时间
2. 确保游戏画面清晰无模糊
3. 调整游戏内 UI 缩放比例为 100%
</details>

<details>
<summary><b>Q: 可以分享给没有 Python 环境的朋友吗？</b></summary>

**A:** 可以！使用 PyInstaller 打包成 .exe 文件后，无需任何环境即可运行。
</details>
